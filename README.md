# SplitScreenMod
Adds support for 4-player split-screen to Batman: Arkham City.

## How to install
Follow these steps to set up the mod for the first time:
1. Install [BmSDK](https://github.com/etkramer/BmSDK#-getting-started)
2. Download the [latest ZIP](https://github.com/etkramer/SplitScreenMod/archive/refs/heads/main.zip) and extract it in `%GameDir%\BmGame\Mods`.
Alternatively, you can clone the repo into the target directory.
That way you can install updates easier.
3. Start the game as usual and press G to add a new player
